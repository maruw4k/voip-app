# voip-app
</br>

Klient voip w oparciu o standard webrtc z możliwością łączenia z natywnymi rozwiązaniami SIP.

</br>

## Funkcje: 
- rozmowy audio, wideo, wiadomości, DTMF
- każda sesja rozmowy w osobnych okienkach
- pokazywanie statystyk połączenia, aktualnej przepustowości na wykresie, oceny jakości głosu w skali MOS
- zakładanie konta, dodawanie znajomych, zarządzanie uprawnieniami
- prezencja 

</br>

## W projekcie wykorzystałem:  
- standard WebRTC
- framework Laravel
- framework AngularJS
- szablon AdminLTE
- bazę danych MySQL
- standard OAuth 
- bilbiotekę SIP.js
- task runner Gulp
- bilbiotekę Chart.js
- bilbiotekę getStats.js
- framework Jasmine
- test runner Karma

</br>

## Instalacja: 

 - composer install
 - npm install
 - skonfigurować plik .env do własnych ustawień 
 - php artisan migrate
 - php artisan db:seed 
 - gulp 
 - php artisan serve --host=0.0.0.0
 - gulp watch
